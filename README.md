# AI-Autocorrect
AI Writing Assistant is a modern web application that helps users improve their writing using a locally hosted Large Language Model (Llama 3.2 via Ollama). It can correct grammar, rewrite text in different tones, and enhance overall readability—all without requiring a paid AI API.
